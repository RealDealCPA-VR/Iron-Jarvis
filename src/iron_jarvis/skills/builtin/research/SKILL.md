# Deep Research

Use this skill when a task calls for rigorous, multi-source investigation rather
than a single quick lookup. The goal is a well-supported answer whose claims can
be traced back to sources.

## Methodology

1. **Frame the question.** Restate the objective in one sentence and list the
   sub-questions that must be answered. If the request is underspecified, note
   the assumptions you are making before starting.
2. **Fan out.** Gather information from several independent sources rather than
   relying on the first hit. Prefer primary sources (official docs, filings,
   datasets) over secondary commentary, and record where each fact came from.
3. **Verify adversarially.** For every load-bearing claim, look for a second
   source that confirms it and actively search for evidence that contradicts it.
   Treat single-source claims as provisional and flag them as such.
4. **Synthesize.** Organize findings around the sub-questions, reconcile
   conflicts explicitly, and separate established facts from inference and
   opinion.
5. **Cite.** Attribute each non-obvious claim to its source so the reader can
   check it. Note the date of time-sensitive information.

## Output

Deliver a concise report: a direct answer up front, supporting findings grouped
by theme, explicit caveats about uncertainty or gaps, and a list of sources.
